Projet informatique de BAB1 2018-2019

Vous trouverez dans cette archive :
- un dossier "src" avec le code source de l'application
- un dossier "lib" avec les librairies java nécessaires au lancement du jeu
- un dossier "images" contenant les différentes images utilisées pour représenter les pièces
- un dossier "doc" contenant la javadoc du projet
- un dossier "saves" où vous trouverez 3 exemples de configurations et où vous pourrez sauvegarder vos propres configurations (au format .position) et vos sauvegardes de parties (au format .dat)
- le fichier "build.xml" permettant de lancer le jeu avec "ant"
- le rapport de projet
- l'énoncé de projet

Les commandes sont :
- ant build : crée un dossier "out" et compile le code
- ant run : lance l'application
- ant clean : suprimme le dossier "out" avec les fichiers compilés
- ant test : lance les tests unitaires
